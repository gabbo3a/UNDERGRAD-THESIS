\section{Dimostrazioni in Lean}

Ecco come appare un blocco di codice strutturato. Nota che puoi usare commenti e simboli speciali senza problemi:

\begin{lstlisting}[
    language=lean, 
    caption={Proprietà associativa dell'unione}]

-- Dimostarzione di prova per formatazione codice lean
example : (p ∨ q) ∨ r ↔ p ∨ (q ∨ r) :=
⟨(fun
    | .inl (.inl hp) => .inl hp
    | .inl (.inr hq) => .inr (.inl hq)
    | .inr hr        => .inr (.inr hr)),
(fun
    | .inl hp        => .inl (.inl hp)
    | .inr (.inl hq) => .inl (.inr hq)
    | .inr (.inr hr) => .inr hr)⟩
\end{lstlisting}
